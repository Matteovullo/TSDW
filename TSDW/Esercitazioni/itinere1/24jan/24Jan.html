Prima parte (richiesta)

Implementare in C o Java, su localhost, un programma server Poll, connection-oriented, che:
  • accetti, su localhost, port 7777, i seguenti messaggi:
    • [G] al quale il server riponderà con un elenco di nomi di animali separati da virgole (sempre lo stesso);
      esempio: cane,gatto,usignolo
    • [Vk] dove k è un intero; ciò costituisce un voto di gradimento per l'animale in posizione k nell'elenco;
      in risposta il server incrementerà il numero di voti espressi per l'animale k; e invierà l'elenco degli animali, accompagnati ciascuno dalla percentuale di voti espressi a suo favore fino a quel momento;
      esempio: cane:50%,gatto:25%,usignolo:25%

      se k non è una posizione valida nell'elenco, il voto si considera nullo e il server risponde con ERROR

  • Il server risponderà ERROR a ogni altro dato ricevuto.

Seconda parte (facoltativa)

Scrivere un programma client che permetta di inviare comandi al server descritto.

NB: il server può comunque essere collaudato con il comando:
telnet localhost 7777

Tempo disponibile per la prova: 60 minuti
